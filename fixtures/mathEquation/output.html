Quadratic formula
x = − b ± b 2 − 4 a c 2 a
Plain-text version

x = ( -b ± √(b² - 4ac) ) / 2a

Mass–energy equivalence
E = m c 2
Plain-text version

E = mc²

Gaussian integral
∫ − ∞ + ∞ e − x 2 d x = π
Plain-text version

∫ (from -∞ to +∞) e^(-x²) dx = √π

Euler’s identity
e i π + 1 = 0
Plain-text version

e^(iπ) + 1 = 0

Summation (Basel problem)
∑ n = 1 ∞ 1 n 2 = π 2 6
Plain-text version

∑ (n=1 to ∞) 1/n² = π² / 6
